Linux는 1991년 리누스 토르발스(Linus Torvalds)가 개발한 Unix 계열의 오픈소스 커널[^kernel]이다.
- 1991년 10월 5일 Linux 커널 0.02 버전이 공개되면서 알려지기 시작했다.
- 초기부터 멀티유저(Multi-user)와 멀티태스킹(Multi-tasking) 환경을 지원했다.
- 이후 멀티프로세서 환경을 지원하도록 발전했다.
- Unix 계열과 유사한 계층형 파일 시스템을 사용하며, POSIX 표준을 상당 부분 지원하여 Unix 계열 환경과 높은 호환성을 가진다.

[^kernel]:커널(Kernel)은 운영체제의 매우 중요한 핵심 부분이다. 하드웨어 자원을 필요한 프로세스에 나눠주고 프로세스 관리, 메모리 제어 등을 수행한다. 예를 들어 웹 브라우저와 VS Code, 음악 프로그램을 동시에 실행하면 커널은 각 프로그램에 CPU 사용 시간과 메모리를 나누어 배정한다.
