## 리눅스 명령어의 구조

#### 명령어 사용법 3가지
- 명령어만 사용
 - #lS, #pwd, #top 등 명령어의 특성에 따라 옵션이나 인자 값이 생략되어도 기본적인 명령어로도 원하는 정보 확인(출력)이 가능하다.
- 명령어와 옵션 사용
 - 옵션은 메인 명령어 뒤에 -와 알파벳을 붙여 사용된다.
 - ex) sudo shutdown -h +60. sudo: 관리자 권한 -h(halt): 시스템 종료# 리눅스 명령어의 구조
- 명령어와 옵션, 인자값 사용
 - ls /boot, ls -a /boot

#### 필수 기초 명령어
- clear: 터미널 화면에 출력된 내용을 지운다.
- man (ls): (ls)특정 명령어에 대한 사용법(매뉴얼)을 확인한다.
- ls(list): 현재 위치 또는 지정한 곳의 목록을 출력한다.
- file: 파일의 종류와 형식을 확인한다.
- find: 주어진 조건의 파일을 찾아 해당 경로를 표시하여 보여준다.
- locate: 주어진 조건의 파일의 위치를 찾아서 보여준다.[^find]
- which: 특정 명령어의 위치를 찾아서 보여준다.
- whereis: 실행파일 및 소스파일 등의 위치를 찾아서 보여준다.
- cat: 파일 내용을 출력해서 화면에 보여줌. 짧은 파일의 내용을 빠르게 확인할 때 사용한다.
- head: 파일 내용을 위 부터 지정된 만큼 보여준다. <-> tail: 파일 내용을 아래 부터 지정된 만큼 보여준다. 기본값은 처음(마지막) 10줄. ex) head(tail) -n 5 file.txt
- more: 파일 내용을 화면 단위로 나누어서 보여준다.
- less: 파일 내용을 화면 단위로 나누어서 보여준다. more보다 기능이 확장되어 위, 아래 이동과 문자열 검색이 가능하다.
- grep: 파일 내용에서 특정 패턴이나 문자열, 정규 표현 식으로 나타낸 단어를 찾아서 보여준다.
- sort: 명령어의 결과나 문서의 내용을 정렬해서 보여준다.
- wc: 순서대로 특정 파일의 줄 수, 단어 수, 바이크 크기를 보여준다.
- cut: 지정된 만큼(특정 필드[^field]) 추출(필드 구분 가능)하여 보여준다.
- split: 하나의 파일을 지정된 크기로 분할한다.
- touch: 빈 파일 생성. touch 파일명, touch -c 파일명: 파일의 시간을 현재 시간으로 변경, touch -t YYYYMMDDhhmm 파일명: 파일의 시간을 날짜 정보로 변경
- mkdir: 디렉토리(폴더) 생성. ex) mkdir -p /test-a/test-b: test-a 디렉토리를 만들고 그 안에 test-b 디렉토리를 만든다.
- cp: 복사. cp test.txt a.txt: test.txt파일을 a.txt로 이름을 변경하여 복사[^dircp]
- mv: 파일, 폴더 이동(원본X). mv {기존 파일명} {바꿀 이름}
- rm: 파일이나 디렉토리를 삭제. rm 파일명 -f: 묻지 않고 삭제(기본은 Y,N). -rf 디렉토리 이름: 디렉토리 삭제(하위 디렉토리 포함)
- >,>> (redirection): 스트림의 방향 조정 및 저장. pwd > test.txt: pwd 내용을 test.txt 파일로 저장. date >> test.txt: date 내용을 기존 test.txt 아래 이어 붙인다.
- alias: 긴 명령어와 옵션을 짧은 단축키처럼 사용할 수 있는 별칭 생성. alias 단축명령어='기존명령어 옵션'. ex) alias cc='ls -al': cc를 입력하면 ls -al과 같은 명령어가 실행된다[^alias].
- ifconfig: 설정된 IP, 서브넷 마스크, 게이트웨이 등을 확인한다.[^subnet][^gateway] 윈도우의 ipconfig


[^find]: find는 실시간, locate는 시스템이 미리 만들어두는 DB를 열어서 읽어오는 것이라, 오늘 방금 만든 파일은 검색이 안될 수 있다.
[^field]: 텍스트 문서에서 띄어쓰기, 쉼표(,), 콜론(:) 같은 특정 기호(구분자)를 기준으로 데이터를 칸칸이 나누었을 때, 그 하나하나의 칸을 필드라고 부른다. ex) 이름,나이,직업 이라는 텍스트 문서에서 cut -d ',' -f 2라고 명령어를 치면 ,로 나눈 것 중 2번째 필드(나이)만 화면에 출력된다.
[^dircp]: mkdir -p /firstdir/seconddir로 두 디렉토리를 만들고 cp -R firstdir seconddir firstdir복사 시 하위 디렉토리와 파일까지 복사한다. <br>* -R 옵션은 지정한 디렉토리뿐만 아니라 그 아래의 모든 하위 데릭토리와 파일까지 포함하여 동일한 명령을 일괄 적용할 때 사용된다.
[^alias]: 생성한 별칭(단축키)는 터미널 종료 시 등록이 사라지며, 영구 등록을 위해서는 편집기를 이용하여 ~/.bashrc 파일에 등록해야 한다. (~는 Home Directory = 윈도우의 C:\Users\사용자명과 같음. .은 숨김 파일. bashrc는 터미널이 실행될 때 읽어들이는 설정 파일.)
[^subnet]: 서브넷 마스크란 IP주소에서 네트워크 ID와 호스트 ID를 구분하기 위해 사용하는 32비트 이진수 비트 마스크이다. 동작 방식은 자신의 IP 주소와 서브넷 마스크를 비트 단위 AND 연산하여 자신이 속한 네트워크 주소를 도출한다.
